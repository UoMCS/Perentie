System Requrements

The following libraries are required to run Perentie.

  • Python 2.6 (Not tested with Python 3-series)
  • PyGTK (Tested under version 2.17, may work on older versions)
  • PySerial
  • Python ELFtools (Optional: for ELF-file support)
  • At least one back-end

Git Repository

The latest development version of Perentie can be found in the (un)official Git repository which you can access using:

git clone https://github.com/mossblaser/Perentie.git
